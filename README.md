# CarRentalAPI
ASP.NET Core Web API created to manage a simple car rental.

## Technologies 
* ASP.NET Core 7
* Entity Framework Core 7
* Microsoft SQL Server
* JWT
* AutoMapper
* FluentValidation
* NLog
* Swagger

## Features
### Account
* Register new user
* Login user

### Car
* Get all cars
* Get specific car
* Add car
* Update car
* Delete car

### Order
* Get all orders
* Get specific order
* Create order
* Update order
* Delete order
